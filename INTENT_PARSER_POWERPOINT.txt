### SLIDE 4: SYSTEM ARCHITECTURE OVERVIEW
**Five-Stage Processing Pipeline**

1. **Input Acquisition**
   - Text input or voice recording with transcription

2. **Structured Extraction**
   - AI-based parsing using OpenAI's structured output models

3. **Location Resolution**
   - Geographic coordinate lookup via Google Maps Geocoding API

4. **Offset Calculation**
   - Mathematical computation of target coordinates from reference points

5. **Visualization**
   - Automatic map display with reference and target locations
